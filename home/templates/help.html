راهنمای استفاده از فاکتورنت

به فاکتورنت خوش آمدید! این سامانه یک ابزار آنلاین برای ساخت و مدیریت فاکتور است و به شما کمک می‌کند فاکتورهای خود را به‌راحتی صادر و ذخیره کنید.

توجه: فاکتورنت فقط یک ابزار تولید فاکتور است و هیچ‌گونه مسئولیتی در قبال پرداخت‌ها یا تعامل بین فروشنده و مشتری ندارد.

چگونه از سایت استفاده کنیم؟

  • ابتدا ثبت‌نام کرده یا وارد حساب کاربری خود شوید.
  • از منوی بالا وارد بخش «داشبورد» شوید.
  • بر روی دکمه «ایجاد فاکتور» کلیک کرده و مشخصات مشتری و آیتم‌های فاکتور را وارد کنید.
  • فاکتور خود را ذخیره کرده و در صورت نیاز فایل PDF را دانلود یا چاپ کنید.
  • در بخش پروفایل می‌توانید نام، ایمیل، آدرس، لوگو و امضای خود را ویرایش کنید.

راهنمای استفاده از API

اگر توسعه‌دهنده هستید، می‌توانید از API فاکتورنت برای مدیریت خودکار فاکتورها استفاده کنید.

پایانه‌های (Endpoints):

  • /invoice/api/customers/ — لیست یا ثبت مشتری
  • /invoice/api/invoices/ — لیست یا ثبت فاکتور
  • /invoice/api/invoice-items/ — آیتم‌های فاکتور

احراز هویت:

برای استفاده از API باید وارد حساب کاربری شوید. فقط فاکتورهایی که خودتان ثبت کرده‌اید برایتان قابل مشاهده یا ویرایش هستند.
